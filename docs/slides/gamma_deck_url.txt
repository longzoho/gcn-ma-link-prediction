Why Gamma was superseded:

The Gamma API on this account is capped at numCards <= 10 per generation.
The spec calls for 22 main slides + 6 appendix = 28 slides total. The Gamma
output was therefore a condensed 10-slide highlights version only. The
python-pptx builder (scripts/build_pptx.py) has no such cap and generates
the full deck with all 11 plot PNGs auto-embedded from results/report/plots/.
